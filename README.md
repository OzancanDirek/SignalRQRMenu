# 🍽️ QR Kodlu Sipariş ve Restoran Yönetim Sistemi (ASP.NET Core + SignalR)

Bu proje, restoranlar için geliştirilmiş **QR kod tabanlı gerçek zamanlı sipariş yönetim sistemidir**.  
Masa bazlı sipariş alma, anlık bildirimler ve admin paneli üzerinden canlı yönetim imkânı sunar.

---

## 🚀 Proje Özeti

Müşteriler QR kod ile menüye erişir, sipariş verir ve bu siparişler **SignalR** kullanılarak anlık olarak admin paneline düşer.  
Admin ve mutfak tarafı siparişleri gerçek zamanlı olarak görür ve durum güncelleyebilir.

---

## 🧩 Özellikler

- 📱 QR kod ile menü erişimi
- 🛒 Masa bazlı sipariş oluşturma
- ⚡ SignalR ile gerçek zamanlı veri akışı
- 👨‍🍳 Mutfak ekranı sipariş takibi
- 📊 Admin paneli üzerinden sipariş yönetimi
- 🪑 Masa durum kontrolü (boş/dolu/meşgul)
- 🔔 Anlık bildirim sistemi
- 🧾 Sipariş detay görüntüleme
